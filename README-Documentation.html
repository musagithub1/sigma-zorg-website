6. üöÄ Deployment Guide

Option 1: Netlify (Recommended for Beginners)

  1. Go to netlify.com
  2. Drag and drop index.html into the deployment area
  3. Your site goes live in seconds!
  4. Get a free HTTPS URL (e.g., sigma-zorg.netlify.app)
‚úÖ Benefits: Free hosting, automatic HTTPS, CDN distribution, instant updates

Option 2: Vercel

  1. Create account at vercel.com
  2. Click "New Project"
  3. Import from GitHub or upload files
  4. Deploy automatically

Option 3: GitHub Pages

  1. Create a GitHub repository
  2. Upload index.html to the repository
  3. Go to Settings ‚Üí Pages
  4. Select branch and save
  5. Access at: https://yourusername.github.io/repo-name/

Option 4: Traditional Web Hosting

  1. Connect to your hosting via FTP (FileZilla, etc.)
  2. Upload index.html to public_html or www folder
  3. Rename to index.html if needed
  4. Access via your domain name
‚ö†Ô∏è Important: Make sure index.html is in the root directory of your web server for the domain to work correctly.

Custom Domain Setup

To use www.sigmazorg.nl:

  1. Purchase domain from registrar (GoDaddy, Namecheap, etc.)
  2. Point DNS A record to hosting IP
  3. Add CNAME record for www subdomain
  4. Enable HTTPS/SSL certificate
  5. Wait 24-48 hours for DNS propagation
